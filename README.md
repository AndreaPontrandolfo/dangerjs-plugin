> this dangerjs plugin decorates your pull-requests when it finds eslint-disable references

## Usage

Install:

```sh
yarn add danger-plugin-eslint-disable-warn --dev
```

At a glance:

```js
// dangerfile.js
import eslintDisableWarn from 'danger-plugin-eslint-disable-warn'

eslintDisableWarn()
```
